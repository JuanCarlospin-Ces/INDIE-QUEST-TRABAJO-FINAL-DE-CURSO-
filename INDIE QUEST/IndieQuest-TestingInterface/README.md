# IndieQuest Testing Interface

React + React Router client used to manually test the IndieQuest API.

👉 Full setup instructions: [../Markdowns/TESTING_INTERFACE_SETUP_GUIDE.md](../Markdowns/TESTING_INTERFACE_SETUP_GUIDE.md)

Quick start:

```powershell
npm install
npm run dev
```

The API must be running on `http://localhost:5063` (or override via `.env`).
